# SingleCellSignalR_companion
Companion repository to SingleCellSignalR (Version 2)

&nbsp;

## Contents

This companion repository to the Bioconductor [SingleCellSignalR]() library,
we illustrate its usage with more flexibility than in the vignette or
the on-line documentation.

In particular, we provide a basic example that illustrates how a single-cell
data set can be analyzed, which may serve as starting point to write your
own code.
